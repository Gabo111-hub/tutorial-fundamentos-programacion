# Tema 2: Métodos (Funciones)

Los métodos (o funciones) son bloques de código que realizan una tarea específica y que podemos "llamar" o ejecutar en cualquier momento. Nos ayudan a mantener el código ordenado, modular y a evitar la repetición.

## 1. Declaración y Retorno
Un método puede recibir datos de entrada (parámetros), procesarlos y devolver un resultado usando la palabra clave `return`. Si el método solo ejecuta una acción y no devuelve ningún valor numérico o de texto, se declara como `void`.

## 2. Modularidad
Al dividir un problema complejo en métodos más pequeños, nuestro programa se vuelve más fácil de leer, mantener y corregir.
